# Project Tactics (Memery Folder)

These tactics are now part of this project direction.

1. Build every gameplay feature as a playable vertical slice first, then expand.
2. Keep the aesthetic cohesive: icy Nordic wilderness with mythic action-RPG atmosphere.
3. Prefer systemic design over scripted moments: weather, temperature, movement, collision, and environment interactions.
4. Preserve responsive controls before adding content complexity.
5. Add one memorable visual signature each milestone (aurora, snow drifts, tavern warmth contrast, etc.).
6. Maintain production-ready structure (modules, clear responsibilities, documented controls).
7. Verify every iteration by running the game and fixing runtime issues immediately.
